# Calculator Application Manual Testing

## Project Overview

This project involves manual testing of a Calculator Application to verify the accuracy and functionality of arithmetic operations and input validations.

## Test Scenarios Covered

* Addition Testing
* Subtraction Testing
* Multiplication Testing
* Division Testing
* Division by Zero Testing
* Decimal Number Testing
* Negative Number Testing
* Large Number Testing
* Empty Input Validation
* Invalid Input Handling

## Testing Type

* Manual Testing
* Functional Testing
* Acceptance Testing
* Test Case Design

## Objective

The objective of this project was to validate the functionality of the calculator application by executing various positive and negative test scenarios and ensuring accurate results.

## Skills Used

* Manual Testing
* Functional Testing
* Test Case Design
* Software Testing
* Quality Assurance

## Outcome

All test cases were executed successfully, and the application's functionality was verified against the expected results.
